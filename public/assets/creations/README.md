# Creation Photos

Place product and gallery photos in this folder.

The starter data in `src/data/creations.ts` references these files:

- `public/assets/creations/dinosaur-birthday-cake.jpg` for the real dinosaur cake photo when available
- `public/assets/creations/farm-birthday-cake.svg`
- `public/assets/creations/birthday-cupcake.svg`
- `public/assets/creations/chocolate-strawberries.svg`
- `public/assets/creations/decorated-sugar-cookies.svg`
- `public/assets/creations/party-favor-boxes.svg`
- `public/assets/creations/holiday-treat-tray.svg`

Replace the SVG placeholders with real JPG or PNG photos, then update the matching `image` paths in `src/data/creations.ts`.

The approved hero/product photo from the design should be saved as:

`public/assets/creations/dinosaur-birthday-cake.jpg`

After that file is in place, update the first item in `src/data/creations.ts` from `farm-birthday-cake.svg` to `dinosaur-birthday-cake.jpg`.
